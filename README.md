# YoBitPumpBot
Yobit Pump Bot

> Python 3+

# How to use

1. Open Terminal or Console (for windows)
2. Enter python ***pump.py*** ***currency***
3. Wait
4. Profit

# Setup config file
***config.py***
```
API_KEY = 'ABC' 
API_SECRET = b'XYZ'
```
# Setup profit settings 

PROFIT_MARKUP, MAX_UNPROFIT and CAN_SPEND
